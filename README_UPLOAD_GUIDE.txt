AGHEC Foundation Website Upload Package
=======================================

Upload this whole package to your GitHub repository root.

Final GitHub structure should be:

/
├── index.html
├── images/
│   ├── aghec-preview.jpg
│   ├── plantation-drive.jpg
│   ├── women-forum.jpg
│   ├── agri-survey.jpg
│   ├── gallery-plantation-1.jpg
│   ├── gallery-women-1.jpg
│   └── gallery-agri-1.jpg
└── publications/
    ├── chintan/
    │   └── index.html
    └── sow-today/
        └── index.html

What to replace
---------------
The images folder currently contains placeholder JPG images. Replace them with your real photos using the exact same filenames.

Recommended photo sizes:
- 1200 x 800 px or bigger
- JPG format
- Under 500 KB each if possible

Important filename rules:
- Use lowercase names exactly as listed.
- Do not use spaces.
- Do not change .jpg to .jpeg or .png unless you also edit index.html paths.

GitHub upload guide
-------------------
1. Open your GitHub website repository.
2. Replace the old index.html with the new index.html from this package.
3. Upload the full images folder.
4. Upload the full publications folder.
5. Click Commit changes.
6. Wait 1-3 minutes for GitHub Pages to update.
7. Open your website in Incognito/Private mode to test without cache.

Publication links
-----------------
The homepage links are branded like your own website:
- /publications/chintan/
- /publications/sow-today/

Contact form
------------
The contact form uses secure HTTPS FormSubmit. The first time it is used, FormSubmit may send a verification email to info@aghec.org.in. Open that email and confirm it once.

Cache tip
---------
If the old website still appears, use Ctrl + Shift + R on desktop or clear browser cache on mobile.
